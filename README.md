# 获取方式

```
# 以下两种版本选择方式示例 2.4.0版本以后

# 集成最新版极验 SDK:
pod 'OneLoginSDK-iOS'

# 集成指定版本极验 SDK，具体版本号可先执行 pod search OneLoginSDK-iOS，根据返回的版本信息自行指定版本
pod 'OneLoginSDK-iOS', '~> 2.4.0'
```
